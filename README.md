# Subay Backend

A scalable backend system built using Node.js, Express.js, and MongoDB. This backend powers the Subay application by handling user authentication, data management, and API services.

## Features

- RESTful API architecture  
- User authentication (JWT-based)  
- Secure login & registration system  
- CRUD operations for application data  
- MongoDB database integration  
- Middleware for authentication and error handling  
- Structured and scalable backend design
